## Robot Basics
* Step 1: Open terminal
* Step 2: Change directory

	 `cd /home/vagrant/Labs/Robo-Security/robo-sec/basics`
	 
* Step 3: Activate virtualenv

	`source venv/bin/activate`	
	
* Step 4: Run a basic robot script
	
	`robot Robo101.robot`  
	
* Step 5: Start flask app

	`start-vul-flask`
	
* Step 6: Run REST API example robot script

	`robot RESTExample.robot`
	
* Step 7: Stop flask app

	`clean-doc`
	
* Step 8: Start wecare app

	`start-wecare`
	
* Step 9: Run Selenium Walkthrough example robot script

	`robot Walkthrough.robot`
	
* Step 10: Stop wecare app

	`clean-doc`
	
* Step 11: Change directory

	 `cd /home/vagrant/Labs/Robo-Security/robo-sec/ci_cd`
	 
* Step 12: Start Node app

	`docker-compose up -d`
	
* Step 13: Run robot scripts as a pipeline

	`robot NodeAppPipeline.robot`	
	
* Step 14: Stop Node app

	`docker-compose down`
	
* Step 15: Change directory

	 `cd /home/vagrant/Labs/Robo-Security/robo-sec/pentest_pipeline`
	 
* Step 16: Start Node app

	`docker-compose up -d`
	
* Step 17: Run robot scripts as a pipeline

	`robot PentestPipeline.robot`	
	
* Step 18: Stop Node app

	`docker-compose down`
	
* Step 19: Deactivate virtualenv

	`deactivate`
